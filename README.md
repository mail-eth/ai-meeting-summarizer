# AI Meeting Summarizer

Automatic meeting notes and action item extraction using Xiaomi MiMo

## 🎯 Features

- Powered by Xiaomi MiMo API
- Simple and intuitive interface
- Real-time processing
- Token consumption: 350K tokens/day

## 🚀 Quick Start

```bash
git clone https://github.com/mail-eth/ai-meeting-summarizer.git
cd ai-meeting-summarizer
npm install
cp .env.example .env
# Add your MIMO_API_KEY
npm start
```

## 📊 Use Case

This project demonstrates practical AI integration using Xiaomi MiMo API for real-world applications.

## 🛠️ Tech Stack

- Xiaomi MiMo API
- Node.js
- Express
- Simple HTML/CSS frontend
